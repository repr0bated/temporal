# Temporal Gemini Context

## Focus Area
**Auditable Workflows**

## Why Index?
Patterns for the "Event Chain" strategy used for regulatory proof-of-change.

## Project Description
Durable execution platform that enables developers to build scalable applications without sacrificing productivity or reliability.

## Key Concepts
- **Workflows**: Durable functions that can run for seconds or years.
- **Activities**: Steps in a workflow that interact with the outside world.
- **Workers**: Processes that execute workflows and activities.
- **History**: Immutable event log of workflow execution.

## Conventions
- **Language**: Go (Server).
- **Style**: Durable execution pattern.
